Write a Python program to print the table of a number entered by the user.


Hello readers in this article I will show you write a Python program to print the table of a number entered by the user.


Program code


n=int(input("please enter any number"))
for i in range(1,11):
print(n,"x",i,"=",n*i)


Output

As you can see with this you can write a Python program to print the table of a number entered by the user.
