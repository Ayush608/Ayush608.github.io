Write a Python program to read today’s date from the user and print how many days are there in the current month.


Hello readers in this article I will show you how to write a Python program to read today’s date from the user and print how many days are there in the current month.


Program code


d=int(input("Please enter today's date"))
m=30
m-=d
print(m,'Days are left')


Output


Please enter today's date15
15 Days are left


As you can see with this you can write a Python program to read today’s date from the user and print how many days are there in the current month.
